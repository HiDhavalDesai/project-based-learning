Welcome to the Number Guessing Game.

Instructions: The computer will randomly generate a number between 1 & 100.
You will enter a number in the text box provided, if you match the number you win, if you don't you will be provided instructions on whether you above or below the number.
Then you can guess again.
You will keep guessing until you win or you decide that have given up. There are two ways to give up, one is by pressing give up, the second way is by looking at the console, we will provide you with the number there.
